---
title: Installation
description: How to install SEEMAP
layout: libdoc/page
---

1. [Download seemap.html](seemap.html){: download=""}
2. Rename if needed.
3. Use it as src attribute of an `iframe` tag or launch it [directly](seemap.html) from web browser.

```html
<iframe src="../seemap.html"></iframe>
<!-- DEMO ONLY -->
<style>
    iframe {
        position: fixed;
        top: 0;
        left: 0;
        width: 100%;
        height: 100%;
        border: none;
    }
</style>
```
